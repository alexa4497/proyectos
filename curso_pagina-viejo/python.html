Introducción a python



Python es un lenguaje de programacion de alto nivel el cual tiene un uso muy amplio en ek area de la programacion,desde desarrollo de aplicaciones web ,hasta una simple calculadora, sistemas de empresas y mas.


Guia de instalacion de python 😎


  1. Entra al siguiente enlace el cual te redireccionara a la pagina oficial de python e iniciar la instalacion
    2. https://www.python.org/downloads/
  2. Ejecuta el instalador

  3. Seleccione install now(instalar ahora)
  4. para comprobar que la instalacion fue exitosa abra el simbolo de sistema o cmd y ejecute el siguiente comando , se debera mostrar la version instalada

    5. python -V

    fin de la guia de instalacion, continua aprendiendo 😉
